## 0.1.3 (2025-03-13)

### Features

- Initial release with core filter DSL
- MongoDB adapter with bracket-to-$ operator transformation
- Schema validation adapters (Zod, Valibot, TypeBox)
- Security utilities (ReDoS protection, field access control)
- Bidirectional query string parsing and building
- Custom operator registry
